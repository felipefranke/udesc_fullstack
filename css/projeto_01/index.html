Pseudo Programação:

IMPUT: palavra usada para receber os dados do usuário

.

PRINT: exibir algo

.

IF (condição): condicional que controla se algo deve ou não ser feito

.

ELSE: cláusula a ser executada caso nenhuma condicional 'if' seja executada

.

LOOP DE X A Y: laço de repetição que irá itinerar de X a Y

.

LOOP X EM Y: laço de repetição que irá iterar X em uma coleção Y

.

WHYLE X: laço de repetição que acontecerá enquanto uma condição for verdadeira
